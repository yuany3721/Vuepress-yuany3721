ssh

编辑~/.ssh/config文件

文件格式：

Host hostalias                          # 关键词
    HostName example.com or ip addr     # 主机地址
    User root                           # 用户名
    IdentityFile ~/.ssh/id_rrrsa        # 认证文件
    Port 22                             # 端口
    ProxyCommand command                # 代理命令

可以使用ssh hostalias替代后面一堆配置选项

没有指定 hostname 的 IdentityFile 默认还是去 id_rsa 做对应
