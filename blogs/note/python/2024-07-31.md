1. Ubuntu
```bash
# 新建虚拟环境
python -m venv venv_name
# 激活虚拟环境
source venv_name/bin/activate
# 停用虚拟环境
deactivate
```
2. Windows
```bash
# 新建虚拟环境
python -m venv venv_name
# 激活虚拟环境
venv_name/Scripts/activate
# 停用虚拟环境
deactivate
```
